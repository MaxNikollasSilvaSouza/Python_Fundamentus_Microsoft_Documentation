# Descrição:

O objetivo desta aplicação é trazer uma lista dos repositórios que determinado usuário possui no GitHub. Será solicitado o nome do usuário de início, e caso ele exista e tenha algum repositório, será realizada a listagem de cada um.

# Como usar o sistema:

Basta selecionar a opção que deseja e fornecer as informações que forem solicitadas.

Será solicitado o nome de um usuário logo no início, e caso ele exista e tenha algum repositório, será realizada a listagem dos repositórios.

Também é possível realizar o push de para algum repositório git basta informar o caminho e preencher o commit corretamente.

Caso queira encerrar a aplicação, basta selecionar a opção 3 quando for possível (o programa pedirá para você escolher uma das opções).
